간단한 웹 기반 계산기입니다.

## 설명

이 프로젝트는 기본적인 산술 연산을 수행하는 웹 계산기입니다.    더하기, 빼기, 곱하기, 나누기 등의 연산을 브라우저에서 직접 수행할 수 있습니다.

## 기술 스택

- JavaScript: 계산 로직 및 인터렉션
- HTML: 페이지 구조
- CSS: 사용자 인터페이스 스타일

## 사용 방법

저장소를 클론한 후 index.html 파일을 웹 브라우저에서 열면 계산기를 사용할 수 있습니다.

README를 이모티콘 없이 텍스트로만 작성해서 수정했습니다. 이제 저장소에서 확인하실 수 있습니다.
