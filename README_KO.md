# 심플 계산기

다크 모드, 다국어 지원, 키보드 지원이 가능한 웹 기반 계산기입니다.

**[English Version](README.md)**

## 기능

- **기본 연산**: 더하기, 빼기, 곱하기, 나누기, 나머지
- **다크 모드**: 밝은 테마와 어두운 테마 전환
- **다국어**: 영어, 한국어, 일본어, 중국어
- **키보드 지원**: 숫자 키, 연산자, Enter로 계산, Backspace로 삭제, Escape로 초기화
- **사운드 효과**: 상호작용 시 음성 피드백
- **반응형 디자인**: 데스크톱과 모바일에서 작동

## 사용 방법

1. 웹 브라우저에서 `index.html` 열기
2. 숫자와 연산자 버튼 클릭하여 계산
3. `=` 또는 Enter 키를 눌러 결과 확인
4. `C`로 초기화, `DEL`로 마지막 숫자 삭제

## 조작 방법

- **버튼**: 클릭하여 숫자와 연산 선택
- **테마 전환**: 🌙/☀️ 버튼을 클릭하여 다크/라이트 모드 전환
- **언어**: 상단의 언어 버튼에서 원하는 언어 선택
- **키보드**: 숫자 키, `+`, `-`, `*`, `/`, Enter, Backspace, Escape 사용 가능

## 기술

- HTML5
- CSS3 (다크 모드 지원)
- JavaScript (Web Audio API를 사용한 사운드 효과)
